Variables y Tipos de Datos

Conceptos esenciales para almacenar y manejar información.

Una variable almacena datos. Ejemplo:
int edad = 25;

Estructuras de Control

Controlan el flujo del programa.

if (x > 10) { console.log("Mayor que 10"); }

Bucles

Repetición de instrucciones.

for (let i = 0; i < 5; i++) { console.log(i); }

Funciones

Bloques reutilizables de código.

function suma(a, b) { return a + b; }
